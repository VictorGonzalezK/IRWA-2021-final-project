# Search Engine with Web Analytics - skeleton project
# IRWA Final Project

This projects contains the startup Flask files for developing a web application.

## Usage

The following project is developed in a python 3 enviroment with the characteristics specified in the requirements.txt file.
For the deployment of the web it is just needed to go to the path where you saved the folder IRWA-2021-final-project-part-4, and proceed in the way described in the following part.
As it is a web application after running the correct file as specified below you will need to open the URL provide at the end in a browser of your choise.
You need to maintain the integrity of the folder and file names and its paths to a correct usage. The database used in the web searcher is stored in the folder database, in case of change of database make sure it follows the same format as the original one provided by the project.

## Starting the Web App

```bash
python -V
# Make sure we use Python 3

python web_app.py
```
The above will start a web server with the application:
```
 * Serving Flask app 'web-app' (lazy loading)
 * Environment: production
   WARNING: This is a development server. Do not use it in a production deployment.
   Use a production WSGI server instead.
 * Debug mode: off
 * Running on http://127.0.0.1:8088/ (Press CTRL+C to quit)
```
